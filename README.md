A parent-friendly website created using Wix AI to help parents identify early signs of developmental disabilities in children and access the right support.

# Features
Early Identification: Guides for Autism, ADHD, speech delay, hearing impairment, and more, with simple illustrations.
Specialists & Doctors: Nearby pediatricians, psychologists, therapists in India, especially Tamil Nadu, with contact info and geolocation support.
Training Centres & Schools: Directories of therapy centres and special education schools in India/Tamil Nadu.
Government Schemes: Disability-specific schemes with eligibility, required documents, and application links.

# Tech Stack
Wix AI
HTML, CSS, JavaScript
Firebase / JSON (for dynamic data)
Google Maps API (for geolocation)

# Purpose
To support parents in early detection, timely intervention, and accessing professional guidance for children with developmental disabilities.
